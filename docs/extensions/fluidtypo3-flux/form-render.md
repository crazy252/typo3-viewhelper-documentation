# <flux:form.render />

Use to render a Flux form as HTML.

### Attributes

| Name | Type | Default | Required | Description |
|:-----|:----:|:-------:|:--------:|:------------|
| form | FluidTYPO3\Flux\Form | `null` | Yes | Form instance to render as HTML |
